Performance Engineering · Production Rescue

API Stability & Bottleneck Resolution

The problem: A mission-critical production API required manual restarts every single day to keep functioning. The development team had normalized this as routine maintenance.

The investigation: Deep profiling and tracing revealed memory leaks under asynchronous load and unmanaged connection pools exhausting system resources overnight.

The fix: Redesigned the task processing architecture with Celery + Azure Service Bus + Redis, deployed via Azure Container Apps and Container Registry, and added API Management for governance and observability.

Result: Zero unplanned restarts for 6+ months. The API is now stable, reliable, and production-grade — without the daily fire drill.
Python FastAPI Celery Azure Service Bus Redis Azure Container Apps Azure Container Registry API Management
